# Blog 프로젝트 

## 기술스택
- JDK 11
- Springboot 2.7.8
- MyBatis
- 테스트 h2 DB
- 프로덕션 배포 MySQL DB
- JSP

## 프로세스

### 1단계
- User
- Board

### 2단계
- Reply
### 3단계
- Love

## 기능정리
### 1단계
- 회원가입, 로그인, 글쓰기, 글목록보기(섬네일제외), 글상세보기, 글삭제, 글수정, 섬네일보기
### 2단계
- 댓글쓰기, 댓글목록보기, 댓글삭제, 프로필사진추가및변경
### 3단계
- 좋아요, 좋아요 취소, 좋아요보기
### 4단계
- ID 중복체크, 회원수정, 페이징
### 5단계
- 검색
